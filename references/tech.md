# 技术搜索

## GitHub（代码仓库）

```bash
python fetch.py "https://github.com/search?q=关键词&type=repositories" -e playwright -w 5
```

## GitHub Issues（问题搜索）

```bash
python fetch.py "https://github.com/search?q=关键词&type=issues" -e playwright -w 5
```

## StackOverflow（技术问答）

```bash
python fetch.py "https://stackoverflow.com/search?q=关键词" -e cffi
```

## V2EX（中文技术社区）

```bash
python fetch.py "https://www.v2ex.com/?q=关键词" -e cffi
```

## Hugging Face（AI 模型）

```bash
python fetch.py "https://huggingface.co/models?search=关键词" -e playwright -w 5
```

## Read the Docs（文档）

```bash
python fetch.py "https://readthedocs.org/search/?q=关键词" -e cffi
```
